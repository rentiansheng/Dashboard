# Dashboard
这是一个数据统计和分析系统。 

# 功能

支持按照周期对数据做计数，求和，平均值及多个字段计算。 

# 数据关系
## 数据分类
- 组织关系
- 数据类型
- 数据实例
## 组织关系
系统的数据通过树状关系组织， 每一个节点可以关联多个数据类型的实例。
